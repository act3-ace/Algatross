[mypy](https://mypy.readthedocs.io/en/stable/) is a python static type-checker. It uses the typehints from your code to confirm correct usage of the codebase.

The easiest way to integrate with ruff is through an IDE such as [VS Code](https://code.visualstudio.com/) and the official [mypy extension](https://marketplace.visualstudio.com/items?itemName=ms-python.mypy-type-checker).

When developing, objects should be typed wherever possible.
